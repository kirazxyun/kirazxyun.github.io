之前不知道有express-generator这样可以直接生成express项目的工具，自己研究了好久了项目框架，绕了不少弯路，简直不要太忧伤。



首先安装express-generator

```Cmd
npm install express-generator -g
```

然后执行

```commonlisp
express test-express
```
